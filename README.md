## Creating My AWS Account

In this step, I'm going to sign up for a new AWS account

### Understanding the root user

I learned that a root user is the user that has the full access and privileges on AWS, and a strong password is needed so it cannot be bypassed and avoid any kind of secuirty breach

## Setting Up Billing and Payment

In this step, I'm going to choose the account plan which is going to be a free tier, enter my contact information and add a payment method

### Why AWS requires a payment method

I learned that AWS requires a payment method because AWS wants to make sure that I am not a bot that is trying to get a free tier acess and to charge me if I passed the limits of the free tier
